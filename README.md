# BLACKDOMAIN AI V3

BLACKDOMAIN AI 是 AI 智能分析平台，提供 AI分析、AI預測、AI建議與AI統計。

## 固定主功能

- 🎲 百家樂AI
- ⚡ 電子AI
- ⚽ 體育AI
- 🎯 539AI
- 👑 VIP中心
- 🌐 黑域官網
- 📞 聯繫管理員

## 可用指令

- 首頁：`黑域AI`、`首頁`、`開始`、`menu`、`選單`
- 百家樂AI：`百家樂`、`百家樂AI`、`baccarat`
- 電子AI：`電子`、`電子AI`、`Electronic`
- 體育AI：`體育`、`體育AI`、`SPORT`、`SPORT AI`
- 539AI：`539`、`539AI`、`今彩539`
- VIP中心：`VIP`、`vip`、`VIP查詢`、`我的VIP`、`會員`
- 黑域官網：`黑域官網`、`官網`
- 聯繫管理員：`聯繫管理員`

## 本機測試

```bash
npm test
```

測試會使用模擬 LINE SDK 驗證所有主要指令、Flex Message、replyMessage、pushMessage、靜態圖片路由與圖片 URL fallback。
